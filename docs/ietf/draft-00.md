               reason:// — Web Agent Reasoning Federation (WARF)
                   draft-ietf-reason-reasoning-artifact-federation-00

Abstract

   This document defines reason://, a protocol and address space for
   sharing compressed structural reasoning artifacts between autonomous
   agents without transferring raw data, models, or sensitive records.
   Artifacts are admitted to the registry only after winning a live
   WARF arbitration round.
